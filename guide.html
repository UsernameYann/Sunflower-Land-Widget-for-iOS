Guide d'installation du widget Sunflower Land sur iPhone

1. Installer le widget avec Scriptable

  1. Téléchargez l'application Scriptable depuis l’App Store.
  2. Ouvrez Scriptable sur votre iPhone.
  3. Créez un nouveau script et copiez-collez le contenu du fichier install_SFL_widget (disponible sur ce dépôt GitHub).
  4. Exécutez le script : il vous demandera votre numéro de ferme et si vous souhaitez activer les notifications.
  5. Le script crée automatiquement le widget MonWidget (ne pas éditer) dans Scriptable.

2. Ajouter le widget sur l'écran d'accueil

  1. Sur l’écran d’accueil, faites un appui long puis appuyez sur le bouton + en haut à gauche.
  2. Recherchez Scriptable dans la liste des widgets.
  3. Ajoutez le widget Scriptable à votre écran.
  4. Dans les options du widget, sélectionnez MonWidget (ne pas éditer).

⚠️ Si le widget n’apparaît pas, ouvrez Scriptable pour vérifier que le script a bien été créé.

3. Fonctionnement du widget

  • Le widget affiche les infos de votre ferme Sunflower Land en temps réel.
  • Si les notifications sont activées, vous recevrez des alertes pour les événements importants.
  • Pour modifier le numéro de ferme ou les notifications, relancez simplement le script d’installation.

Pour toute question, consultez la page GitHub du projet.
