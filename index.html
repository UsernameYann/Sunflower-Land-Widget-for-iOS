⚡ With Shortcuts

3️⃣ Open Scriptable

Open the Scriptable app on your iPhone. Tap the + button to create a new script.

4️⃣ Paste and save

Paste the generated code, name it "SFL_widget.js", and save. Add the widget to your home screen!

3️⃣ Install "Install_SFL_Widget" (optional)

How to use it:

  1. Click Add Shortcut right below:
➕ Add Shortcut
  1. Add the Shortcut.
  2. The Shortcuts app will open, tap "Install_SFL_Widget".
⚠️ Required permissions on first launch
  • Allow file saving.
  • Also allow file deletion.

Once validated, the file will be in the Scriptable app. You can close Shortcuts.

4️⃣ Open and run the script

Open Scriptable on your iPhone and tap SFL_widget in the list to run it. You can close Scriptable.

5️⃣ How to Add the Widget to Your Home Screen

  1. Go to your iPhone Home Screen.
  2. Long press and tap the + button (top left).
  3. Search for Scriptable in the widget list.
  4. Add the Scriptable widget, then select SFL_widget in the widget settings.
Important: If the widget does not appear, open the Scriptable app to sync your scripts.

  4. Create new script: Tap the + button to create a new script
